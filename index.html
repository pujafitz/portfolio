FEATURED PROJECTS

Goldman Stacks
A STOCK MARKET SIMULATOR
  • Live Site
  • GitHub Repo

A real-time stock-trading simulation, combining live market data and news with server-side processing to create a robust environment to test out real trading strategies.

Webpages are generated with a server-side Express templating engine and a single page load involves multiple API calls, database queries, and calculations to bring all the data together.

Featured Technologies: Node.js, Express, PostgreSQL, and AWS

GitHub Finder
A GitHub Search App
  • Live Site
  • GitHub Repo

A project created in React using the GitHub API to search for GitHub users. After a search inquiry, the user can then choose any of the resulting GitHub profiles displayed to gather additional information on that particular GitHub user.

Coming Soon: Each user's GitHub repositories displayed on their "More" page, as well as functionality to search within a particular user's repos.

Featured Technologies: React, Node.js, Express, and AWS

Space Force Prime
A SPACE SHOOTER GAME
  • GitHub Repo

Space Force Prime is a Python3-only application designed as a single-player top-down scrolling space shooter game.

It's the year 2069 (nice!), and President Barron Trump's U.S. Space Force program is in ruins. Amazon's Cyborg CEO, Jeff Bezos 4.0, just bought out the derelict military branch. As a newly-hired delivery human, it's your job to deliver packages to Prime customers while defending your ship, crew, and cargo. Good luck ...and watch out for asteroids!

Featured Technologies: Python and pygame

Let's Get Quizzical
A FUN TRIVIA GAME
  • Live Site
  • GitHub Repo

Let's Get Quizzical is an 80s-styled trivia game created using Frontend technologies. Single-player (for now), a user can choose the number of questions they'd like, and from which category. "Surprise Me!" serves up questions from randomized categories.

Coming Soon: Two-player Mode with buzzing in and separate player scores

Featured Technologies: HTML, CSS, JavaScript, and AWS
